## 📐 Design Principles

- **Distributed Authority** — humans and agents collaborate based on risk level and context
- **JSON-First** — all tools emit structured outputs for agent consumption
- **Static-First** — dashboards and reports require zero backend for visitors
- **Hermes-Native** — tools and skills are callable from Telegram via Hermes Agent
- **Multi-Chain** — Celo, Solana, Base, Polygon — chain-agnostic where it matters
