# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any Atlas Nexus repository, **do not open a public issue**.

Send details to: **atlasnexus.ops@proton.me**

We will respond within 48 hours. Please include:
- Repository and version affected
- Steps to reproduce
- Potential impact

## Supported Versions

| Repository | Supported |
|-----------|-----------|
| ai-lance / ai2work | ✅ Active |
| All others | Best effort |

## Disclosure

We follow coordinated disclosure. After a fix is released, we will credit the reporter (unless they prefer to remain anonymous).
